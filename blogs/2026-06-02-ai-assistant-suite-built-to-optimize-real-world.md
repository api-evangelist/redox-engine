Redox unveiled three new AI-powered assistants designed to streamline integration management: a Config Modifier Assistant for generating data mapping schemas, a Log Summary Assistant that explains processing events in plain language, and a Payload Summary Assistant that translates raw transaction data into readable formats. These tools aim to simplify integration management workflows while enabling non-technical users to resolve routine issues independently.
